Usando links externos

Usamos links externos adicionando eles através da tecla 'a' desse jeito: Youtube

Entretanto, se deixarmos o link desse jeito, ele irá fechar a nossa página para abrir a outro. Podemos evitar isso digitando o seguinte logo após o link: 'target: '_blank'' e 'rel: 'external''. Por exemplo:

YouTube

Links Internos de uma página

É possível criar links internos dentro da nossa pagina que nos direciona para uma nova página. Basta criarmos um novo documento html. Basta clicar no 'a' novamente e colocar o diretório do arquivo pressionando ctrl + space.
